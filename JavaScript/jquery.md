# jQuery

## General

- jQuery is a JavaScript library that uses an API to improve document traversal manipulation.
  - Affects
    - DOM elements
    - Event handling
    - Animations
    - Ajax
  - Initiation
    - jQuery must be initiated with the following function:
      ```javascript
      // ES5
      
      $(document).ready(function () {
        //code block
      });
      
      // ES6
      
      $(function() {
        // code block
      });
      ```
